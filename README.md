# A graph builder and visualiser

This is a simple graph builder (uses a simple adjacency list implimentation in OOP style). I plan to use this in another, more ambitious project. I needed a "backend" implementation of graphs, and a way to visualise it in canvas. Thats why i built this. 

I used a repulsion and spring based, numerical analysis styled, physics algorithm. I tried not to look up any existing algorithms for this. But, i am very satisfied with the way the algorithm looks. I mean it was a physics based idea, so clearly it looks like a bunch of subatomic particles... I do plan to research better algorithms, and implement it. It will be more useful for the other ideas i have with this stuff acting as the core "engine".
